📌 왜 지금 다운로드해야 하나요?

2025년 6월, 바이낸스에 7억원 상당의 가상자산을 보유한 A씨는 해외금융계좌 신고를 하지 않아 7,000만원(10%)의 과태료를 부과받았습니다. A씨는 "해외 거래소는 국세청이 모를 것"이라고 생각했지만, 국제 정보 교환 협정(CARF)을 통해 모든 정보가 노출되었습니다.

2026년 3월 현재, 2027년 1월 1일부터 해외 거래소 수익도 국내와 동일하게 과세됩니다. 바이낸스, 바이비트 등 모든 해외 거래소는 CARF(Crypto-Asset Reporting Framework)를 통해 한국 국세청에 거래 정보를 자동으로 제공하게 됩니다.

문제는 바이낸스 웹사이트에서는 최대 90일(3개월)치 거래내역만 다운로드할 수 있다는 점입니다. 2027년 세금 신고 시 2027년 1월 1일부터 12월 31일까지 전체 내역이 필요하지만, 90일 제한 때문에 매 3개월마다 다운로드해야 합니다. 이 가이드는 3년치 전체 내역을 한 번에 다운로드하는 방법을 알려드립니다.

💡 세대별 이해 돕기

20대 (Z세대) – "넷플릭스 시청 기록 다운로드와 비슷해요"

넷플릭스에서 시청 기록을 CSV로 다운로드하듯, 바이낸스도 거래 내역을 CSV로 받을 수 있습니다. 차이점은 바이낸스는 90일 제한이 있어서 API를 사용하거나 고객센터에 요청해야 전체 내역을 받을 수 있다는 점입니다. 마치 넷플릭스 전체 시청 기록을 받으려면 고객센터에 문의해야 하는 것과 비슷합니다.

30~40대 (밀레니얼) – "은행 거래내역 조회와 동일해요"

은행에서 계좌 거래내역을 엑셀로 다운로드하는 것처럼, 바이낸스도 Trade History에서 CSV 파일을 다운로드할 수 있습니다. 단, 은행은 전체 기간 조회가 가능하지만 바이낸스는 90일 제한이 있습니다. 전체 내역은 API 키를 생성하거나 고객센터에 이메일로 요청해야 합니다. 마치 은행 통장 사본을 발급받는 것처럼 공식 요청이 필요합니다.

50~60대 (베이비부머) – "증권사 매매 확인서와 비슷합니다"

증권사에서 주식 매매 확인서를 발급받는 것처럼, 바이낸스에서도 거래 증명을 위한 CSV 파일을 다운로드할 수 있습니다. 차이점은 증권사는 전체 기간 발급이 가능하지만, 바이낸스는 90일 제한이 있다는 점입니다. 3년치 전체 내역은 API(프로그램 인터페이스)를 사용하거나 고객센터에 이메일로 요청해야 합니다. 신고 방법은 홈택스에서 증권사 거래와 동일하게 처리됩니다.

🛠️ 바이낸스 거래내역 다운로드 5가지 방법

방법 1: 바이낸스 웹에서 90일치 다운로드 (난이도: ★☆☆☆☆ | 소요 시간: 3분)

📍 가장 간단하지만 최대 90일 제한

단계별 가이드:

  1. 바이낸스 웹사이트 로그인
  2. 우측 상단 프로필 아이콘 클릭 → 'Orders (주문 내역)' 선택
  3. 'Trade History (거래 내역)' 탭 클릭
  4. 우측 상단 'Export Trade History (거래내역 내보내기)' 버튼 클릭
  5. 기간 선택: 최대 90일 (예: 2026-01-01 ~ 2026-03-31)
  6. 파일 형식: CSV 또는 XLSX 선택
  7. 다운로드 완료 (파일명: Trade_History_2026-01-01_2026-03-31.csv)

⚠️ 주의사항:

  • 최대 90일 제한 → 2027년 전체 신고 시 4번 다운로드 필요 (1~3월, 4~6월, 7~9월, 10~12월)
  • Spot (현물), Futures (선물), Margin (마진) 거래는 각각 별도 다운로드
  • CSV 파일에는 거래 날짜, 코인명, 수량, 가격, 수수료 포함

방법 2: 바이낸스 API로 3년치 전체 다운로드 (난이도: ★★★☆☆ | 소요 시간: 30분)

📍 프로그래밍 지식 없어도 Python 스크립트 복사하면 가능

사전 준비:

  • Python 3.7 이상 설치 (무료 다운로드: python.org)
  • 바이낸스 API 키 생성 (아래 단계 참고)

API 키 생성 방법:

  1. 바이낸스 로그인 → 프로필 → 'API Management (API 관리)'
  2. 'Create API (API 생성)' 클릭
  3. API Key Label: "Tax Report" (임의 입력)
  4. 보안 인증 (이메일 + SMS 코드 입력)
  5. API Key와 Secret Key 복사 (절대 유출 금지!)
  6. 권한 설정: 'Enable Reading (읽기 전용)'만 체크 (거래 권한은 체크 해제)

Python 스크립트 실행:

  1. GitHub에서 무료 스크립트 다운로드: binance-exports
  2. 터미널 (Windows: CMD, Mac: Terminal) 열기
  3. 명령어 입력: 
    pip install binance-exports
binance-exports --api-key YOUR_API_KEY --secret YOUR_SECRET_KEY --start 2024-01-01 --end 2026-12-31
  4. CSV 파일 자동 생성 (파일명: binance_trades_2024-2026.csv)

⚠️ 주의사항:

  • API Key는 절대 타인에게 공유 금지 (계정 해킹 위험)
  • 읽기 전용(Read-Only) 권한만 부여할 것
  • 사용 후 API Key 비활성화 권장

📌 출처: GitHub – binance-exports (공식 Python 스크립트)

방법 3: 바이낸스 Transaction History 다운로드 (난이도: ★★☆☆☆ | 소요 시간: 10분)

📍 입출금 + 거래 + 수수료 전체 내역 포함 (최대 1년)

단계별 가이드:

  1. 바이낸스 로그인 → 'Wallet (지갑)' 클릭
  2. 'Transaction History (거래 내역)' 선택
  3. 우측 상단 'Generate All Statements (모든 명세서 생성)' 클릭
  4. 데이터 유형: 'Asset History – Transaction History' 선택
  5. 기간 선택: 최대 1년 (예: 2025-01-01 ~ 2025-12-31)
  6. 파일 형식: 'Excel' 선택
  7. 'Generate (생성)' 클릭 → 이메일로 발송 (약 10~30분 소요)
  8. 이메일에서 다운로드 링크 클릭 → Excel 파일 다운로드

✅ 포함 내역:

  • Deposit (입금), Withdrawal (출금)
  • Buy/Sell (매수/매도)
  • Trading Fees (거래 수수료)
  • Staking Rewards (스테이킹 보상)
  • Referral Commission (추천 수수료)

⚠️ 주의사항:

  • 최대 1년 제한 → 3년치는 3번 다운로드 필요
  • 파일 크기가 큰 경우 이메일 발송이 지연될 수 있음

방법 4: 바이낸스 고객센터에 3년치 전체 요청 (난이도: ★☆☆☆☆ | 소요 시간: 3~5일)

📍 가장 확실하지만 시간 소요

이메일 요청 템플릿:

To: support@binance.com
Subject: Request for Complete Trade History (2024-2026)

Dear Binance Support Team,

I am writing to request my complete trade history for tax reporting purposes.

- Account Email: your_email@example.com
- User ID: [귀하의 User ID]
- Requested Period: 2024-01-01 to 2026-12-31
- File Format: CSV or Excel

Please include all transactions:
- Spot Trading (현물 거래)
- Futures Trading (선물 거래)
- Margin Trading (마진 거래)
- Deposits & Withdrawals (입출금)
- Staking Rewards (스테이킹 보상)

Thank you for your assistance.

Best regards,
[귀하의 이름]

처리 절차:

  1. 위 템플릿 복사 → support@binance.com으로 이메일 발송
  2. 신원 확인 요청 (KYC 서류: 신분증 사본)
  3. 3~5일 후 CSV 파일 이메일 발송

✅ 장점:

  • 프로그래밍 지식 불필요
  • 전체 기간 내역을 한 번에 받을 수 있음
  • 공식 증명 서류로 인정 (세무사 제출 가능)

⚠️ 주의사항:

  • 답변까지 3~5일 소요 (급한 경우 비추천)
  • 영어로 작성해야 답변이 빠름

방법 5: 세금 계산 도구 연동 (난이도: ★★☆☆☆ | 소요 시간: 15분)

📍 자동 계산 + CSV 다운로드 + 세금 신고서 작성까지 한 번에

추천 도구:

  • CryptoTax.cloud (한국어 지원, 바이낸스 API 자동 연동)
  • Koinly (영어, 바이낸스 + 100개 거래소 지원)
  • CoinTracker (영어, 미국 세금 중심)

사용 방법:

  1. CryptoTax.cloud 회원가입 (무료)
  2. '거래소 연동' → 'Binance' 선택
  3. API Key + Secret Key 입력 (읽기 전용)
  4. 자동으로 3년치 전체 내역 불러오기
  5. 세금 계산 자동 완료 (과세 소득, 납부 세액)
  6. CSV 다운로드 + 홈택스 신고서 자동 작성

✅ 장점:

  • 수동 계산 불필요 (손익 자동 계산)
  • 여러 거래소 통합 관리 (바이낸스 + 업비트 + 코인원)
  • 홈택스 신고서 자동 작성

💰 비용:

  • 무료 플랜: 거래 100건까지
  • 프로 플랜: 월 19,900원 (거래 무제한)

🚨 3가지 치명적 실수와 예방법

❌ 실수 1: "바이낸스는 국세청이 모르겠지?"

→ 2027년부터 CARF로 자동 추적. OECD 가상자산 정보 교환 협정(CARF)에 따라 2027년부터 바이낸스, 바이비트 등 모든 해외 거래소는 한국 국세청에 거래 정보를 자동으로 제공합니다. 2025년 6월 사례처럼 7억원 보유자는 7,000만원 과태료를 받았습니다.

❌ 실수 2: "5억원은 월평균이겠지?"

→ 월말 기준 하루라도 5억원 초과 시 신고 대상. 예: 2025년 12월 31일 하루만 5.2억원 → 2026년 6월 30일까지 신고 필수. 월평균이 아닌 월말 최대값이 기준입니다.

❌ 실수 3: "90일씩 다운로드하면 되지"

→ 2027년 5월 신고 시 4번 다운로드 + 수동 병합 필요. 바이낸스는 90일 제한이므로 2027년 전체 내역은 1~3월, 4~6월, 7~9월, 10~12월 4번 다운로드 후 Excel에서 수동 병합해야 합니다. API 사용이 훨씬 효율적입니다.

❓ 자주 묻는 질문 (FAQ)

Q1. 바이낸스 거래내역은 최대 몇 개월치까지 다운로드 가능한가요?

A. 바이낸스 웹에서는 최대 90일(3개월)치만 다운로드 가능합니다. 3년치 전체 내역은 API 사용 또는 고객센터 이메일 요청으로만 받을 수 있습니다.

Q2. 바이낸스 5억원 초과 시 신고해야 하나요?

A. 네. 월말 기준 하루라도 5억원 초과 시 다음 해 6월 30일까지 해외금융계좌 신고 필수. 미신고 시 최대 20% 과태료 부과. 예: 2025년 12월 31일 5.2억원 → 2026년 6월 30일까지 신고.

Q3. 바이낸스 수익도 2027년부터 세금 내야 하나요?

A. 네. 2027년 1월 1일 이후 발생 수익은 연 250만원 초과분에 22% 과세. 해외 거래소도 OECD CARF(가상자산 정보 교환 협정)를 통해 국세청에 정보 제공 예정.

Q4. 바이낸스 API Key는 안전한가요?

A. 읽기 전용(Read-Only) 권한만 부여하면 안전합니다. 거래 권한(Enable Trading)은 절대 체크하지 마세요. API Key는 절대 타인에게 공유하지 말고, 사용 후 비활성화하는 것이 안전합니다.

Q5. 바이낸스와 업비트를 같이 사용하는데 따로 신고하나요?

A. ❌ 통합 신고 필수. 모든 거래소의 수익을 합산해서 한 번에 신고해야 합니다. 예: 바이낸스 +700만원, 업비트 -200만원 → 순수익 500만원 - 250만원 = 250만원 × 22% = 55만원 납부.

🎯 이해도 퀴즈

문제: 영희는 2025년 12월 31일에 바이낸스에 5.2억원 상당의 비트코인을 보유했고, 2026년 1월 1일에 전량 매도하여 4.8억원으로 줄었습니다. 영희는 해외금융계좌 신고를 해야 하나요?

정답:

  • ✅ 신고 필수. 2025년 12월 31일 기준 5.2억원 → 2026년 6월 30일까지 신고
  • 신고 기준은 월말 기준 하루라도 5억원 초과 여부
  • 2026년 1월 1일 4.8억원으로 줄었더라도 2025년 12월 31일 기준으로 신고 대상

📌 포인트: 해외금융계좌 신고는 월평균이 아닌 월말 최대값이 기준입니다. 하루라도 5억원을 초과하면 신고해야 합니다.
