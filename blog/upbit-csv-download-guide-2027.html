📌 업비트 거래내역 CSV 다운로드란?

업비트 거래내역 CSV 다운로드란 업비트 거래소에서 발생한 모든 매수·매도·입출금 기록을 CSV(Comma-Separated Values) 형식의 파일로 추출하는 것을 의미합니다. 2027년 가상자산 과세 시행을 앞두고, 정확한 세금 계산을 위해 반드시 확보해야 하는 핵심 데이터입니다.

쉽게 말해, 업비트 거래내역 CSV는 여러분의 코인 투자 '가계부'입니다. 언제, 얼마에, 어떤 코인을 샀고 팔았는지 모든 기록이 담겨 있죠. 이 데이터가 없으면? 국세청에 "나는 이만큼 벌었어요"라고 말해도 증빙이 없는 셈입니다.

📊 2026년 3월 현재 상황

국세청에 따르면 2027년 1월 1일부터 가상자산 양도·대여 소득에 대해 22%(지방세 포함) 세율이 적용됩니다. 연간 250만원 기본공제 후 초과분에 과세되며, 정확한 취득가액 증빙이 없으면 양도가액의 최대 50%만 필요경비로 인정받습니다.

🚨 취득가액 증빙 못하면 벌어지는 일

예시: 1,000만원에 산 비트코인을 2,000만원에 팔았다면?

  • 증빙 있을 때: 양도차익 1,000만원 - 250만원 = 750만원 × 22% = 165만원 납부
  • 증빙 없을 때: 2,000만원 × 50% = 1,000만원 필요경비 → 1,000만원 - 250만원 = 750만원 × 22% = 165만원 납부 (운 좋은 경우)
  • 최악의 경우: 필요경비 인정 비율이 낮아지면 세금 폭탄!

🛠️ 업비트 거래내역 CSV 다운로드 방법 3가지

현재 업비트에서 거래내역을 CSV로 받는 방법은 크게 3가지입니다. 각 방법의 난이도와 장단점을 비교해 드리겠습니다.

방법 1: 업비트 고객센터 요청 (가장 쉬움)

1 고객센터 요청 절차

  1. 업비트 앱/웹 → 고객센터 접속
  2. 1:1 문의 클릭
  3. 문의 유형: "거래내역 관련" 선택
  4. 내용 작성: "전체 거래내역 엑셀(CSV) 파일 발급 요청드립니다. 기간: 20XX년 XX월 ~ 현재"
  5. 본인 확인 절차 후 이메일로 수신 (3~5영업일 소요)

장점: 가장 간편하고 공식적인 방법
단점: 처리 시간이 오래 걸림, 형식이 정해져 있음

방법 2: 업비트 Open API 활용 (중급자)

2 API 키 발급 및 데이터 추출

  1. 업비트 로그인 → 마이페이지 → Open API 관리
  2. API 키 발급 (자산조회, 주문조회 권한 체크)
  3. Access Key와 Secret Key 안전하게 보관
  4. Python/JavaScript 등으로 거래내역 조회 API 호출
  5. 결과를 CSV 파일로 저장

장점: 실시간 데이터, 원하는 형식으로 가공 가능
단점: 프로그래밍 지식 필요, API 호출 제한 있음

💡 API 활용 팁

업비트 개발자센터(docs.upbit.com)에서 상세 문서를 확인하세요. GET /v1/orders 엔드포인트로 주문 내역을, GET /v1/deposits와 GET /v1/withdraws로 입출금 내역을 조회할 수 있습니다.

방법 3: PDF 다운로드 후 변환 (대안)

3 PDF를 엑셀로 변환하기

  1. 업비트 앱/웹 → 고객센터 → 증명서 발급
  2. 거래내역서 발급 선택
  3. 원하는 기간 설정 후 PDF 다운로드
  4. Adobe Acrobat 또는 온라인 변환 도구로 CSV/Excel 변환
  5. 변환된 데이터 정확성 검토 (필수!)

장점: 즉시 발급 가능
단점: 변환 과정에서 데이터 오류 가능성, 수동 검토 필요

❌ 실수 1: 입출금 내역 누락

문제: 매수·매도 내역만 다운로드하고 입출금 기록을 빠뜨림

결과: 다른 거래소나 개인지갑으로 이동한 코인의 취득가액 증빙 불가

해결: 거래내역 + 입금내역 + 출금내역 모두 다운로드

❌ 실수 2: 기간 설정 오류

문제: 최근 1년만 다운로드하고 과거 데이터 누락

결과: 2017~2020년에 매수한 코인의 취득가액 증빙 불가

해결: 가입일부터 현재까지 전체 기간 선택

❌ 실수 3: 원본 파일 미보관

문제: 다운로드 후 가공만 하고 원본 삭제

결과: 세무조사 시 원본 증빙 불가

해결: 원본 파일은 별도 폴더에 날짜별로 보관 (클라우드 백업 권장)

❌ 실수 4: 다른 거래소 데이터 미수집

문제: 업비트만 다운로드하고 빗썸, 코인원, 해외거래소 누락

결과: 전체 손익 계산 불가, 손익통산 혜택 못 받음

해결: 사용한 모든 거래소의 거래내역 수집

❌ 실수 5: 마감일 직전 준비

문제: 2028년 5월에 급하게 데이터 수집

결과: 시간 부족, 오류 발생, 가산세 위험

해결: 지금 당장 시작! 분기별로 정기 다운로드

❓ 자주 묻는 질문 (FAQ)

Q1: 업비트에서 CSV 파일을 직접 다운로드할 수 있나요?

A: 현재 업비트 공식 서비스는 거래내역서를 PDF 형식으로만 제공합니다. CSV나 엑셀 파일이 필요한 경우 고객센터(1588-5682)에 별도로 요청해야 합니다. Open API를 활용하면 직접 데이터를 추출할 수도 있습니다.

Q2: 거래내역 보관 기간은 얼마나 되나요?

A: 업비트는 법적 요구에 따라 거래내역을 최소 5년간 보관합니다. 하지만 더 오래된 데이터의 경우 시스템 변경 등으로 조회가 어려울 수 있으므로, 정기적으로 다운로드하여 별도 보관하는 것을 강력히 권장합니다.

Q3: 2027년 이전에 산 코인은 어떻게 계산하나요?

A: 소득세법에 따르면 2027년 1월 1일 전에 보유한 가상자산의 취득가액은 "2026년 12월 31일 시가"와 "실제 취득가액" 중 큰 금액으로 계산합니다. 따라서 과거 취득가액을 증빙해두면 유리할 수 있습니다.

Q4: 여러 거래소를 사용했는데 어떻게 하나요?

A: 모든 거래소(업비트, 빗썸, 코인원, 해외거래소 등)의 거래내역을 각각 다운로드한 후, 세금 계산기에 함께 업로드하면 됩니다. 손익통산은 전체 거래를 합산해 계산하므로, 모든 거래소 데이터가 필요합니다.

Q5: API 키 발급이 위험하지 않나요?

A: API 키 발급 시 "출금 권한"은 절대 체크하지 마세요. "자산조회"와 "주문조회" 권한만 부여하면 안전하게 데이터만 조회할 수 있습니다. 키는 타인에게 절대 공유하지 말고 안전하게 보관하세요.
