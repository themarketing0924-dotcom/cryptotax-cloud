  sitemap   ✅ 최신화 완료 (2026-05-04)

① 구글 서치 콘솔 등록 (가장 중요!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

접속: https://search.google.com/search-console/

STEP 1: "속성 추가" 클릭
STEP 2: "URL 접두어" 선택 → https://cryptotax.cloud/ 입력
STEP 3: "HTML 태그" 인증 방법 선택
STEP 4: 표시된 content 값 복사
         예: <meta name="google-site-verification" content="AbCdEf123456">
STEP 5: index.html 열기 → 아래 줄 찾아서 content 값 교체:
         【GOOGLE_VERIFICATION_CODE】 → 복사한 코드로 교체
STEP 6: 깃허브에 push → "확인" 클릭

STEP 7 (등록 완료 후): Sitemap 제출
  서치 콘솔 → 왼쪽 메뉴 "Sitemaps" → 아래 입력:
    sitemap.xml
  → "제출" 클릭

  효과: 제출 후 1~3일 내 구글 수집 시작
        새 글 올릴 때마다 자동 크롤링 (robots.txt 덕분)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
② 빙(Bing) 웹마스터 등록
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

접속: https://www.bing.com/webmasters/

STEP 1: Microsoft 계정으로 로그인
STEP 2: "내 사이트 추가" → https://cryptotax.cloud 입력
STEP 3: "XML Sitemap" 옵션 선택
         → https://cryptotax.cloud/sitemap.xml 입력
STEP 4: "HTML 메타 태그" 방법 선택 → content 값 복사
STEP 5: index.html에서 아래 줄 찾아 교체:
         【BING_VERIFICATION_CODE】 → 복사한 코드로 교체
STEP 6: 깃허브 push → "확인" 클릭

  특이사항: 빙에 등록하면 ChatGPT 검색(Bing 기반)에도 노출됨!
            AI 검색 트래픽 확보를 위해 반드시 등록 권장

③ 네이버 서치어드바이저 확인
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

접속: https://searchadvisor.naver.com/

현재 상태: index.html에 인증 태그 있음 ✅
           content="4c5c9c813d6e2c25e421f8e77af396957bbb11d1"

확인할 것:
  STEP 1: 서치어드바이저 로그인
  STEP 2: "사이트 목록" → cryptotax.cloud 상태 확인
  STEP 3: "요청" → "사이트맵 제출" → sitemap.xml 제출:
           https://cryptotax.cloud/sitemap.xml
  STEP 4: "요청" → "수집 요청" → 홈페이지 URL 수집 요청

  네이버 수집 주기: 1~2주 (구글보다 느림, 꾸준한 발행이 더 중요)

④ 다음(Daum)/카카오 등록
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

접속: https://webmaster.daum.net/

STEP 1: 카카오 계정으로 로그인
STEP 2: "사이트 등록" → https://cryptotax.cloud 입력
STEP 3: "확인 파일" 다운로드 방식 선택
         다운받은 HTML 파일 → 사이트 루트에 업로드 후 push
STEP 4: Sitemap 제출:
         https://cryptotax.cloud/sitemap.xml

  참고: 다음은 카카오와 통합됨. 다음 뉴스·카카오톡 공유 시
        콘텐츠가 잘 수집되려면 등록해두는 것이 유리

⑤ 구글 인덱싱 촉진 — URL 검사 도구
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

새 글을 발행할 때마다:
  서치 콘솔 → "URL 검사" → 새 글 URL 입력 → "색인 생성 요청"
  → 24~48시간 내 구글 수집 (sitemap만으로는 1~2주 걸릴 수 있음)

  예: https://cryptotax.cloud/blog/coin-tax-abolition-bill-2027.html
      → 서치 콘솔에서 수동 색인 요청

  - 색인 생성된 페이지 수 (Coverage)
  - 클릭수·노출수·평균 순위 (Performance)
  - 모바일 사용성 오류 (Mobile Usability)

네이버 서치어드바이저:
  - 수집 현황 (수집된 페이지 수)
  - 검색 통계 (유입 키워드)

목표:
  1개월 → 전체 페이지 색인 완료
  3개월 → TIER 2 키워드 상위 20위 진입
  6개월 → TIER 1 키워드 상위 10위 진입
