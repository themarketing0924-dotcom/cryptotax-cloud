CryptoTax.cloud 블로그 포스트 생성 프롬프트
클로드(Claude) / 잰스파크(Jasper)에 그대로 복사·붙여넣기
================================================================

아래 지시사항을 모두 지켜서 한국어 블로그 포스트를 작성해줘.

━━━━━━━━━━━━━━━━━━━━━━━━━━━
주제: 【여기에 주제 입력】
예시:
  - "2027년 비트코인 ETF 투자자 세금 신고 방법"
  - "해외 거래소 바이낸스 이용자 국내 세금 신고 의무"
  - "가상자산 증여·상속 세금 계산법"
━━━━━━━━━━━━━━━━━━━━━━━━━━━

[작성 규칙]

1. 타겟 독자: 한국 가상자산(코인) 투자자, 30~40대, 세금이 걱정되지만
   전문 지식은 없는 일반인. 어렵지 않게, 친절하게 설명할 것.

2. 글 구조 (반드시 준수):
   - 핵심 요약 (danger-box): 2~3문장, 이 글의 핵심을 한눈에
   - 섹션 1: 개념 설명 + 표 (항목 vs 내용)
   - 섹션 2: 계산 방법 or 구체적 사례 + 공식
   - 섹션 3: 실전 전략 or 주의사항 + highlight 박스
   - 섹션 4: 자주 묻는 질문 Q&A (3개)
   - CTA: 세금 계산기로 유도하는 문구

3. SEO 요구사항:
   - 제목: 30~55자, 주요 키워드 앞에 배치
   - 메타 설명: 120~155자
   - 주요 키워드를 제목, 첫 문단, 소제목에 자연스럽게 포함
   - FAQ 3개의 질문은 실제로 사람들이 구글에 검색하는 문장으로

4. 출력 형식:
   아래 항목들을 채워서 텍스트로 출력해줘
   (HTML 코드 자체를 출력하지 말고, 항목별로 분리해서):

   [파일명] 영어소문자-하이픈.html (예: bitcoin-etf-tax-2027.html)
   [제목] 한글 제목
   [배지] 예: 🔥 2025년 5월 최신
   [히어로 부제목] 1~2줄
   [CTA 버튼] 버튼 텍스트
   [핵심 요약] 2~3문장
   [섹션1 제목] / [섹션1 본문] / [섹션1 표: 항목|내용 형식]
   [섹션2 제목] / [섹션2 본문] / [공식]
   [섹션3 제목] / [섹션3 본문] / [강조 포인트]
   [FAQ 질문1] / [답변1]
   [FAQ 질문2] / [답변2]
   [FAQ 질문3] / [답변3]
   [CTA 헤드라인] / [CTA 서브텍스트]
   [메타 설명]
   [키워드 5개]
   [Article 태그 2개]
   [이미지1 설명 (영어 프롬프트)] 16:9 비율 이미지 생성용
   [이미지2 설명 (영어 프롬프트)]
   [이미지3 설명 (영어 프롬프트)]

5. 한국 세법 정확도:
   - 2027년 가상자산 과세: 연간 수익 250만 원 초과분에 22% (소득세 20%+지방세 2%)
   - 기본공제: 연 250만 원
   - 손익통산: 동일 과세연도 내 허용, 이월공제 불가
   - 신고: 2027년 귀속소득 → 2028년 5월 종합소득세 신고 (분리과세)
   - 위 내용과 모순되는 내용 작성 금지

      이미지 사이즈: 1200×630px (16:9)
□ 4. blog/index.html 열어서 카드 1개 추가 (아래 참고)
