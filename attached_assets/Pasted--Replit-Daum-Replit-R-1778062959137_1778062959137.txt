## **📋 Replit에 전달할 명령 프롬프트**

```
아래 내용을 index.html (cryptotax.cloud 메인 페이지)에 적용해줘.
Daum 검색등록 심사 통과를 위한 수정이야. 총 6가지 항목이야.

────────────────────────────────────────
✅ 수정 1. [이메일 수집 섹션 완전 제거]
────────────────────────────────────────
아래 섹션 전체를 HTML에서 삭제해줘.
"무료 절세 가이드 PDF 받기" 제목을 포함한
이메일 입력 폼 전체 블록을 찾아서 제거해.

삭제할 키워드로 찾기:
- "무료 절세 가이드 PDF 받기"
- "이메일 등록 시 위 3가지를 무료로 보내드립니다"
- "스팸 없음 · 언제든 수신 거부 가능"
- "무료 PDF 즉시 발송"
위 텍스트가 포함된 section 또는 div 블록 전체 삭제.

────────────────────────────────────────
✅ 수정 2. [상단 Breaking 뉴스 스크롤 배너 제거]
────────────────────────────────────────
상단에 빨간 "BREAKING" 뱃지와 함께
좌우로 스크롤되는 텍스트 배너가 있어.
class="breaking" 인 div 전체를 삭제해줘.

────────────────────────────────────────
✅ 수정 3. [Hero 섹션 과장 통계 수치 수정]
────────────────────────────────────────
히어로 섹션의 hstat(통계 카드) 4개 중
아래 항목을 수정해줘.

변경 전 → 변경 후:
- "1,326만" 한국 코인 투자자 → "가상자산" 투자자 (숫자 제거)
- "10초" 즉시 계산 → 삭제하거나 "간편" 계산으로 변경

────────────────────────────────────────
✅ 수정 4. [Hero CTA 버튼 문구 완화]
────────────────────────────────────────
히어로 영역의 cta-primary 버튼 문구를 찾아서:
"10초 만에 계산하세요" 또는 이와 유사한 클릭 유도 문구를
"세금 계산하기" 또는 "계산기 바로가기" 로 변경해줘.

────────────────────────────────────────
✅ 수정 5. [파트너 배너 섹션 축소]
────────────────────────────────────────
현재 thecoinlabs.com 파트너 링크/배너가
메인 페이지에 3곳 이상 반복 노출되고 있어.
아래 기준으로 줄여줘:

- 메인 페이지 내 thecoinlabs.com 링크를
  footer 1곳만 남기고 나머지 모두 제거
- class가 "ad-slide", "coinlabs-logo" 등
  광고 슬라이드 형태인 블록에서
  thecoinlabs 관련 슬라이드 카드 제거

────────────────────────────────────────
✅ 수정 6. [D-day 카운트다운 섹션 표현 완화]
────────────────────────────────────────
class="urgency" 인 섹션의 h2 제목을 찾아서:
현재: "⏰ 2027년 가상자산 과세까지 남은 시간"
변경: "2027년 가상자산 과세 일정 안내"
로 수정하고, 이모지(⏰)는 제거해줘.

urgency 섹션 내부의 카운트다운 숫자 위젯은 유지해도 되지만
섹션 전체를 히어로 최상단이 아닌
계산기 섹션 아래쪽으로 순서를 이동시켜줘.

────────────────────────────────────────
🚫 건드리지 말아야 할 것:
────────────────────────────────────────
- 계산기 12종 기능 전체 (tools-grid)
- FAQ 섹션
- 네이버/구글/Bing 메타 태그
- robots meta 태그
- canonical 태그
- sitemap 관련 설정
- footer 저작권 및 기본 링크
```

## **✅ 수정 완료 후 Daum 재신청 체크리스트**

수정이 끝나면 아래 항목을 직접 눈으로 확인 후 재신청하세요.

| 번호 | 확인 항목 | 상태 |
|------|-----------|------|
| 1 | 이메일 수집 폼이 메인 페이지에서 보이지 않는가 | ☐ |
| 2 | "PDF 무료 발송", "스팸 없음" 문구가 사라졌는가 | ☐ |
| 3 | 상단 빨간 BREAKING 스크롤 배너가 없는가 | ☐ |
| 4 | "1,326만" 같은 과장 수치가 제거되었는가 | ☐ |
| 5 | thecoinlabs 파트너 링크가 1곳 이하인가 | ☐ |
| 6 | 버튼 문구에 "즉시", "10초" 같은 클릭 유도 표현이 없는가 | ☐ |
| 7 | 사이트 핵심 기능(계산기)은 정상 동작하는가 | ☐ |
