<section class="section" id="faq">

  <div class="section-title">

    <h2>❓ 자주 묻는 질문</h2>

    <p>코린이들이 가장 많이 찾는 질문 모음</p>

  </div>

  <div class="faq-list">

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">코인 세금은 언제부터 내야 하나요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">현행 법률 기준으로 <strong>2027년 1월 1일</strong>부터 가상자산 양도소득세가 부과됩니다. 다만 2026년 3월 국민의힘이 폐지 법안을 발의하여 국회 통과 여부에 따라 변경될 수 있습니다. 통과 전까지는 현행 기준(22%)으로 미리 파악해두는 것이 현명합니다.</div>

    </div>

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">코인 세율 22%는 어떻게 계산되나요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">소득세 20% + 지방소득세 2% = 총 22%입니다. 연간 수익에서 기본공제 <strong>250만원</strong>을 뺀 금액(과세표준)에 적용됩니다. 수익이 250만원 이하면 세금이 0원입니다.</div>

    </div>

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">업비트에서 거래하면 세금 신고 어떻게 하나요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">업비트 앱 → 더보기 → 거래내역 다운로드(CSV)로 1년치 거래 내역을 받고, 다음 해 5월 국세청 홈택스에서 종합소득세 신고 시 가상자산 항목에 입력합니다. 정확한 신고 방법은 국세청 공식 안내를 참고하세요.</div>

    </div>

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">손실이 났을 때도 세금을 내나요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">아닙니다. 같은 해에 이익 난 코인과 손실 난 코인의 손익을 합산(통산)합니다. 예를 들어 비트코인으로 1,000만원 수익, 이더리움으로 500만원 손실이면 순 수익 500만원에 세금이 적용됩니다. 연말에 손실 코인을 매도하면 세금을 줄일 수 있습니다.</div>

    </div>

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">물타기와 DCA의 차이가 무엇인가요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">물타기는 <strong>하락 중인 코인에 감정적으로 추가 매수</strong>하는 것이고, DCA(달러 비용 평균법)는 <strong>가격과 무관하게 정해진 날짜에 정해진 금액을 꾸준히 투자</strong>하는 전략입니다. 물타기는 손실이 심화될 수 있는 반면, DCA는 장기적으로 평균 매수단가를 낮추는 안정적인 전략입니다.</div>

    </div>

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">사토시가 뭔가요? 1만원으로 비트코인을 살 수 있나요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">사토시(Satoshi)는 비트코인의 최소 단위입니다. 1 비트코인 = <strong>1억 사토시</strong>입니다. 비트코인이 1억원이라도 1만원으로 0.0001 BTC(=1만 사토시)를 살 수 있습니다. 업비트에서는 최소 5,000원부터 매수 가능합니다.</div>

    </div>

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">김치 프리미엄이란 무엇인가요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">국내 거래소의 코인 가격이 해외 거래소보다 높게 형성되는 현상입니다. 예를 들어 비트코인이 해외에서 1억원인데 업비트에서 1억 500만원이면 김치 프리미엄 5%입니다. 원화 자금 유입 증가, 국내 투자 열기 과열 시 발생하며 역사적으로 20~30%까지 올라간 적도 있습니다.</div>

    </div>

    <div class="faq-item">

      <div class="faq-q" onclick="toggleFaq(this)">이 계산기의 결과가 실제 세금과 다를 수 있나요? <span class="faq-arrow">▼</span></div>

      <div class="faq-a">네. 본 사이트의 계산 결과는 <strong>참고용 추정치</strong>이며 실제 세금 신고에는 거래소별 취득가액 산정 방식, 기타 소득 합산 여부 등 복잡한 요소가 추가됩니다. 정확한 세금 계산은 세무사 상담 또는 국세청 공식 채널을 이용하시기 바랍니다.</div>

    </div>

// ── 세금 계산 ──

function calcTax() {

  const buy  = parseFloat(document.getElementById('buyAmt').value)  || 0;

  const sell = parseFloat(document.getElementById('sellAmt').value) || 0;

  const fee  = parseFloat(document.getElementById('feeAmt').value)  || 0;


  if(!buy || !sell) {

    alert('매수금액과 매도금액을 입력해주세요.');

    return;

  }


  const profit   = sell - buy;

  const deduct   = 2500000;

  const taxBase  = Math.max(0, profit - deduct - fee);

  const incomeTax = taxBase * 0.20;

  const localTax  = taxBase * 0.02;

  const totalTax  = incomeTax + localTax;

  const effRate   = profit > 0 ? (totalTax / profit * 100).toFixed(2) : 0;

  const netProfit = profit - totalTax;


  document.getElementById('r-sell').textContent    = fmt(sell);

  document.getElementById('r-buy').textContent     = fmt(buy);

  document.getElementById('r-profit').textContent  = fmt(profit);

  document.getElementById('r-fee').textContent     = fee > 0 ? '-' + fmt(fee) : '0원';

  document.getElementById('r-base').textContent    = fmt(taxBase);

  document.getElementById('r-income').textContent  = fmt(incomeTax);

  document.getElementById('r-local').textContent   = fmt(localTax);

  document.getElementById('r-total').textContent   = fmt(totalTax);

  document.getElementById('r-rate').textContent    = effRate + '%';

  document.getElementById('r-net').textContent     = fmt(netProfit);


  // 색상 적용

  const profitEl = document.getElementById('r-profit');

  profitEl.style.color = profit >= 0 ? '#3fb950' : '#f85149';


  const box = document.getElementById('resultBox');

  box.style.display = 'block';

  box.scrollIntoView({behavior:'smooth', block:'nearest'});

}

    else totalLoss += v;

  const netTotal = totalProfit + totalLoss;

  const deduct   = 2500000;

  document.getElementById('net-total').textContent  = fmt(netTotal);
